# PNGC Registry

A registry of GitHub repositories from labs affiliated with the Penn Neurodegeneration Genomics Center (PNGC).

## Purpose

This registry helps PNGC-affiliated labs share and discover computational tools, pipelines, and resources related to neurodegeneration genomics research.

## How to Register Your Repository

1. Check that your repository meets the [requirements](#requirements)
2. Click on **Issues** â†’ **New Issue**
3. Select **"Register a Repository"**
4. Fill out the template
5. Submit the issue
6. A maintainer will review and add your repository to the registry

## Requirements

### Required Information

- **Lab/Group Name**: Your lab or research group name
- **Repository Link**: Full GitHub URL
- **Description**: 1-2 sentence description of what the repository does
- **Primary Contact**: Name + email or GitHub handle
- **Topic Tags**: Select from approved tags (see below)
- **License**: Specify license type or indicate "no license yet"
- **If Published**: 
  - Citation (paper title + journal/year)
  - DOI or PubMed link

### Recommended

- **Status**: active / maintenance / archived
- **README Quality**: Should include what it does and how to run it
- **Funding Acknowledgment**: Optional but encouraged
